# calculadora-basica-java
## Funcionalidades 
- Soma 
- Subtração
- Multiplicação
- Divisão 
- Menu interativo
- ## Tecnologias e conceitos aplicados - Java (JDK) - Estruturas condicionais (`if / else`) - Métodos estáticos - Entrada de dados com `Scanner` - Loop de repetição

## Como executar 
1. Compile o arquivo: javac CalculadoraBasica.java

2. Execute o arquivo: java CalculadoraBasica
